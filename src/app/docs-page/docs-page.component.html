Introduction

Get started with Angular 5 boilerplate that utilize service bus architecture, that helps you to build responsive, bulletproof projects on the web with the boilerplate that is great from the smallest to atomic projects.

Quick start

Looking to quickly use that boilerplate for your project? Just clone the boilerplate from GitHub. All you need is NPM to install the packages and you are ready to go!

To start with the boilerplate, just follow the steps:

  1. Clone the boilerplate from GitHub
  2. Install node package manager or Yarn
  3. If you have NPM go to the root of the project and type in the console npm installor if you are using Yarn just type yarn
  4. When it finish, if you are using NPM type in the consolenpm run startor if you are using Yarn just type yarn start
  5. If the end of the build tells youwebpack: Compiled successfully. you are ready to go!

Project structure

To work with that boilerplate you need to know the basics of Angular. Here is good source to learn of the basics of Angular

  • src - Contains all of the front-end code
    • app - Contains all of the scripts
      • core - Contains all of the core services for the app
      • feedback - Contains all of the directives, components and services for user interaction for events and errors
      • services - Contains all of the services that are used for data transformation of the fetched from back-end
      • shared - Contains all of the directives, components and services that are used in multiple locations
    • assets - Contains all of the fonts, images and other assets.
      • fonts - Contains all of the fonts
      • images - Contains all of the images
    • enviroments - Contains all of the enviroment variables.
      • enviroment.local.ts - Contains all of the enviroment variables for the local development
      • enviroment.prod.ts - Contains all of the enviroment variables for the prod development
      • enviroment.ts - Contains all of the enviroment variables for when enviroment is not selected
    • styles.scss - Contains all of the global styling.
    • index.html - Contains the basic HTML page of the app.
  • .angular-cli.json.json - Contains all of the build procedures.
  • package.json - Contains all of the used packages.
